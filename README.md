# MoeCli

Pink-themed multi-provider coding CLI.

## Install

```bash
npm i -g @moetanorg/moecli
```

## Run

```bash
moecli
```

## Requirements

- Node.js 22+

## Features

- OpenAI Responses
- OpenAI Compatible
- Anthropic
- Amazon Bedrock
- Google Gemini
- Local tools, browser integration, and sub-agents

## Development

```bash
npm install
npm run dev
```

## Publish

```bash
npm publish
```
